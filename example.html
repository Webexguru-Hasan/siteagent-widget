Quick Start

Add the widget to your website in just 2 steps:

Step 1: Configuration

Add this script before the widget script:

<script>
  window.SiteAgentConfig = {
    apiKey: 'YOUR_API_KEY',              // Get from SiteAgent dashboard
    serverUrl: 'https://api.siteagent.com',  // Your SiteAgent server
    tenantId: 'YOUR_TENANT_ID'           // Your tenant identifier
  };
</script>

Step 2: Load Widget

Add the widget script:

<script src="https://cdn.example.com/siteagent.min.js"></script>
✨ That's it! The floating chat button will appear in the bottom-right corner of your page.

Configuration Options

Customize the widget behavior with these options:

Option Type Description
apiKey string (required) Your API key from SiteAgent dashboard
serverUrl string SiteAgent server URL (default: http://localhost:3000)
tenantId string Your tenant ID (default: 'default-tenant')

API Endpoints

The widget communicates with your SiteAgent server via JSON-RPC 2.0:

Endpoint

POST {serverUrl}/mcp

Headers

Content-Type: application/json
Authorization: Bearer {apiKey}

Tools Called

  • detect_context — Analyzes page URL and user intent
  • search_content — Finds relevant content from your site
  • get_user_context — Loads conversation history
  • save_user_context — Saves messages to history

Troubleshooting

Widget not showing?

1. Check browser console for errors (F12)

2. Verify SiteAgentConfig is set before loading the script

3. Ensure API key is valid and server URL is correct

Messages not sending?

1. Check that your SiteAgent server is running

2. Verify network request in browser DevTools (Network tab)

3. Check CORS headers from server

Chat history not persisting?

1. Ensure browser allows localStorage

2. Check if your server database is connected

3. Verify session_id is being saved
